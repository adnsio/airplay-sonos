AirPlay server for Sonos devices

![alt text](https://s14.postimg.org/4za0g7pkx/Screen_Shot_2016_09_15_at_01_08_45.png "iTunes")

#### Install and run
```bash
git clone git@github.com:alessiodionisi/airplay-sonos.git
cd airplay-sonos
npm install
npm start
```
